This folder contains of tiny .asm files that are enough to teach people the basic Syntax of x86_64 NASM assembly. Read every single .asm file carefully, this folder isn't enough to teach you the entire assembly language, but it is absolutely enough to make you able to read, write and understand assembly enough for projects like game engines, low latency systems, etc. Note: This assembly is 86_64 NASM assembly that runs on user space of Ubuntu (a Linux distribution).
